# Masterprompt — Who Knows Enterprise

Baue das Carry OS Modul `Who Knows Enterprise` als produktionsnahes System für eine Streetwear-Marke.

## Ziel

Ein System für Content, Shop, Drops und Automatisierung:

- UGC Video Center
- Landing Page Builder
- Shop System
- Drop Launch System
- Waitlist und CRM
- Agent Control Center
- Analytics und Learning Engine

## Architekturregeln

- TypeScript-first
- Zod Validierung
- zentrale Error Handler
- API unter `/api/v1/who-knows/*`
- Mockdaten im MVP
- später Provider Adapter für Shopify, Stripe, TikTok, Instagram, YouTube, ElevenLabs, Runway, Canva, CapCut
- Mobile-first UI
- Dark luxury streetwear aesthetic
